# AirSense

AirSense is an AI-powered contactless human monitoring system built with **ESP32-S3 Wi-Fi CSI** and a native **macOS application**.

Instead of wearable devices, AirSense analyzes Wi-Fi Channel State Information (CSI) to understand human activity inside a room.

The goal is to create a privacy-friendly indoor sensing platform capable of monitoring human presence, movement, breathing, heart rate, sleep, and emergency situations using AI.

---

## Vision

AirSense aims to transform ordinary Wi-Fi signals into a real-time human sensing system.

Future capabilities include:

- Human Presence Detection
- Multi-Person Tracking
- Motion Recognition
- Heart Rate Estimation
- Breathing Rate Estimation
- Sleep Monitoring
- Fall Detection
- Emergency Detection
- Live CSI Visualization
- Mobile Notifications
- Historical Reports
